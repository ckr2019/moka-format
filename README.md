# Moka format

自动 sort imports 的 VS Code 插件，Moka 前端团队内部使用

## Usage

1. Search "Moka format" in VS Code marketplace and install it

2. Select all imports, press Cmd + Shift + P / Win + SHIFT + P, and then use command "Moka format"
